## Design Spec
![Image](./images/figma-spec.png)

## Requirements
- Build from scratch
- Fill in own information
- Separate components for:
    - Info (photo, name, buttons, etc)
        - img
        - h1 name
        - h3 title
        - p website 
        - two buttons
            - email
            - linked in 
            - maybe add a github
    - About
        - h2 About title
        - p about paragraph 
    - Interests
        - h2 Interests title
        - p Interests paragraph
    - Footer (social icons)
        - different color
        - four icons
            - twitter
            - facebook
            - instagram
            - github
